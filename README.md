# pull-tab card anatomy — a muslin

A d.school-style muslin (deliberately rough, seams showing) for penelope's card/drawer
anatomy: file-folder pull tabs, two-zone poles (exterior fold-out drawer + interior
sections around a Now line), tab-travel with fold-over close, drop-opens-pinned.

**Browse it live:** the GitHub Pages link on this repo. Pull the tabs, hit expand, drag
the fake row. It's built to be torn apart — the banner and debug labels are the point,
not oversights. Rulings-so-far live in NOTES.md.
